# Survey Data
This project aims to reduce the dimensionality of neighborhood survey data. The dataset includes 7 neighborhood dimensions used to assess the neighborhood environment and its association with cardiovascular outcomes.  
**Our goal** is to reduce the dimensionality of the survey data and further evaluate the relationship between neighborhood environment and cardiovascular outcomes.  
We first generate empirical Bayes estimates of individual-level mean neighborhood perception scores, adjusted for age and sex, using a linear mixed-effects model.   
We then compare Principal Component Analysis (PCA) and Factor Analysis (FA) for dimensionality reduction.  

-  Ecometric_analysis.Rmd contains the code for this analysis
-  Ecometric_analysis.pdf is the report generated using this Rmd file.
